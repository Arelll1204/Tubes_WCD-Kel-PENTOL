## Cara Menjalankan

1. Clone repositori ini
2. Buka `index.html` di browser
3. Klik "Mulai Plan" untuk ke halaman Planner

> Tidak perlu server — bisa langsung dibuka sebagai file HTML biasa.

## Struktur File

```
menu-planner/
├── index.html       # Landing page
├── planner.html     # Halaman Planner
├── style.css        # CSS eksternal (satu file untuk kedua halaman)
├── script.js        # JavaScript DOM manipulation
└── README.md
```

## Fitur

- ✅ Landing page bergaya marketing dengan hero, cara kerja, menu favorit, testimoni, CTA
- ✅ Tabel 26 menu lengkap dengan kolom: Gambar, Nama, Kategori, Kalori, Harga, Checkbox
- ✅ Panel hasil real-time: total kalori, total harga, checklist 5 kategori, badge "Menu Seimbang!"
- ✅ Filter menu berdasarkan kategori
- ✅ Responsif (mobile-friendly dengan bottom sheet)
- ✅ Logic badge "Menu Seimbang" murni JavaScript (tidak hardcode)

## Teknologi

- HTML5 semantik
- CSS3 eksternal (variabel CSS, flexbox, grid, media queries)
- Vanilla JavaScript (DOM manipulation, addEventListener, createElement, classList)
- Google Fonts (Plus Jakarta Sans + Lora)
